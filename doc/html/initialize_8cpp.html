Parse an XML and energy file to obtain atom and interaction information.

Parse an XML and energy file to obtain atom and interaction information. Returns 0 if successful, non-zero if failure. Operates in a "cascade" between ranks so that each processor (if MPI is used) opens and initializes from the file in order. Returns 0 if successful, else integer flag for failure. Does domain decomposition.

Parameters
[in]xml_filenameName of coordinate file to open and read.
[in]energy_filenameName of file containing bonds, pair potential parameters, etc.
[in]nprocsNumber of processors total.
[in]rankRank of this process.
[in,out]sysPointer to System object to store its information at.
